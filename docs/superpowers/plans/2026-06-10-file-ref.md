# @ 文件引用实现计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** composer 输入 `@` 弹出工作区文件浮层,过滤后选中把 `@相对路径 ` 插入消息文本(不注入内容)。

**Architecture:** 后端新增 `GET /files/index`(`LocalFileStore.walk` 递归列文件,跳符号链接,limit 2000);前端纯函数 `fileRef.ts` 派生 @ 词与过滤,Composer 跟踪 caret 复用 `SlashPalette` 渲染,@ 浮层压过斜杠面板。

**Tech Stack:** FastAPI + pathlib rglob;React 19 + react-query(`["fileIndex", userId]`,staleTime 30s)+ vitest。

**Spec:** `docs/superpowers/specs/2026-06-10-file-ref-design.md`

### Task 2: 前端 `fileRef.ts` 纯函数

**Files:**
- Create: `frontend/src/fileRef.ts`
- Test: `frontend/src/fileRef.test.ts`

- [ ] 失败测试:`atTokenAt`(句首 `@`、空白后 `@`、邮箱 `a@b` 不触发、中文 query、caret 在词中间只取 `[start,caret)`、无 `@` → null、`@` 后跟空白 → query 截止);`filterPaths`(大小写不敏感子串、保序、max 截断、空 query 全量)
- [ ] 实现:
  - `atTokenAt(text, caret)`:从 caret 向左扫到空白/行首得词首 `s`;`text[s] === "@"` 且(`s===0` 或 `text[s-1]` 是空白)→ `{ start: s, query: text.slice(s+1, caret) }`(词内不含空白由扫描保证;query 不含 `@`,含则 null,防 `a@@b`)
  - `filterPaths(paths, query, max=20)`:`q=query.toLowerCase()`,`paths.filter(p => p.toLowerCase().includes(q)).slice(0, max)`
- [ ] vitest 绿 → commit `feat(frontend): fileRef 纯函数(@ 词解析与路径过滤)`

**Files:**

- Modify: `frontend/src/components/Composer.tsx`

- [ ] 失败测试(mock `api.indexFiles` → `["src/app.py", "docs/读我.md", "README.md"]`):
  - 打 `@` → listbox 列文件(title=basename,hint=完整路径)
  - 打 `@读` → 只剩中文命中项
  - Enter → 文本变 `@docs/读我.md `(选中项路径 + 尾空格),浮层关
  - Esc → 浮层关,同词不再弹;改 query 再弹
  - `@` 活跃时输入以 `/` 开头文本(如 `/model @x`)→ 文件浮层优先(覆盖参数模式建议)
  - 无匹配 query → 无浮层;IME composing 回车不选中(既有守卫路径)
- [ ] Composer 实现:
  - `const [caret, setCaret] = useState(0)`;textarea `onChange`/`onSelect` 里 `setCaret(e.currentTarget.selectionStart ?? 0)`
  - `const [atDismissed, setAtDismissed] = useState<string | null>(null)`(记 Esc 时的 token start+query 键;token 变化自动失效)
  - `const token = atTokenAt(text, caret)`;`useQuery({ queryKey: ["fileIndex", userId], queryFn: api.indexFiles, enabled: !!userId && !!token, staleTime: 30_000 })`
  - token 活跃且未 dismissed:`filterPaths(paths, token.query)` → entries(title=basename,hint=path,exec=替换 `[token.start, caret)` 为 `@path ` + `setCaret`/`setSel(0)`,用 ref 把焦点与 selectionRange 设回 textarea)
  - 文件 entries 非空 → 覆盖斜杠 entries(@ 优先);键盘路由复用现有 paletteOpen 分支;Esc 在 @ 模式下 `setAtDismissed(key)` 而非 `setDismissed(true)`
- [ ] vitest 绿 + `npm run lint` → commit `feat(frontend): composer @ 文件引用浮层`
